# Colors

The palette is exposed as CSS custom properties on `:root`. Always reference the
tokens (`var(--color-*)`) rather than hard-coding hex values.

| Token | Hex | Role |
|---|---|---|
| `--color-velvet` | `#08060c` | App void background |
| `--color-obsidian` | `#121017` | Grimoire panels / cards back |
| `--color-ink` | `#110e13` | Woodblock ink — text on gold, hard shadows |
| `--color-gold` | `#d9b461` | Gold leaf — primary accent, borders, CTAs |
| `--color-gold-dark` | `#8c6a23` | Antique gold — quieter borders/rules |
| `--color-crimson` | `#a62424` | Blood/wine accent — alerts, chalices, "doomed" |
| `--color-paper` | `#f5f2eb` | Aged card stock — body text, card faces |
| `--color-paper-dark` | `#e8e3d5` | Card-stock shadow |
| `--color-shadow` | `rgba(0,0,0,.7)` | Generic shadow |

Idiom: gold-on-ink and ink-on-gold are the two primary contrasts; crimson is
reserved for danger / loss / judgement.
